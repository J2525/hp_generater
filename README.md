# HpGenerater

This gem makes template of "html&css"

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hp_generater'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hp_generater

## Usage

```ruby
$ hp_generater xxx
```
make directory "xxx" , and make file "index.html & sample.css" in this directory.
